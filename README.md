<h1>Analisador Léxico</h1>

<h3>O que é um Compilador? </h3>
<p>Um compilador é um programa que lê
um programa escrito numa linguagem fonte e o traduz
num programa numa lingugem alvo. </p>

<h3> As fases de um compilador:</h3>
<li>
<strong> Fase de Análise:</strong> está associado se o programa está
escrito corretamente de acordo com a regras da linguagem. 
Subdividido em análise léxica, análise sintática e análise semântica.
</li>
<li>
<strong>Fase de Síntese:</strong> visa gerar o código alvo. 
</li>

<h2>A Análise Léxica</h2>
<p> Está relacinada a verificação se os nomes das entidades
estão corretos. </p>

<h2>Objetivo do trabalho acadêmico</h2>
<p>Escolher uma linguagem de programação e,
atrvés do JFlex construir um analisador léxico. Todos
os tokens envolvidos nos comandos mais basicos devem
estar corretamente relacionados. </p>
<li> A linguagem escolida foi: C#</li>
